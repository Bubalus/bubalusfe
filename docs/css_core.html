BubalusFE Web 前端開發工具集 —— 沒錯，雖然有那麽多英文，但這并不是什么高級貨！

BubalusFE，是一組就像遛狗大嬸一樣常見的 WEB 前端開發工具集，它所提供的功能也好像總是色迷迷盯著遛狗大嬸的散步大爺一樣普通，蹲在路邊虐殺螞蟻的小女孩也說：“它根本沒什麽了不起！”

它誕生于2007年的秋天，一個懶惰的傢伙爲了減輕工作壓力而拼湊出了它的原型。這個傢伙真是個會耍小聰明的懶鬼，使用這組工具后，他的工作用時節省了一半，但他還是向領導上報跟以前一樣多的工時，從此他感覺工作變得輕鬆了許多。真是可喜可賀！可喜可賀！

工具類 一些用於輔助布局和呈現的類

浮動元素

使用 .fl-le 類實現 float: left 的效果；
使用 .fl-ri 類實現 float: right 的效果 

<div class="fl-le">...</div>
<div class="fl-ri">...</div>

清除浮動

使用 .clear 類實現 清除浮動 的功能 

<div class="clear">...</div>

顯現和隱藏

使用 .hide 類實現 display: none 的效果；
使用 .show 類實現 display: block 的效果；
使用 .invisible 類實現 visibility: hidden 的效果； 

<div class="hide">...</div>
<div class="show">...</div>
<div class="invisible">...</div>

響應系統 爲響應式應用提供基礎

通過媒體查詢 @media 來對各種分辨率的屏幕或打印機提供專有的樣式。

響應類

類名 手機 768px 以下 平板電腦 768px - 979px 桌面電腦 980px 以上
.visible-sm 可見的 不可見的 不可見的
.visible-md 不可見的 可見的 不可見的
.visible-lg 不可見的 不可見的 可見的
.hidden-sm 不可見的 可見的 可見的
.hidden-md 可見的 不可見的 可見的
.hidden-lg 可見的 可見的 不可見的

打印類

類名 瀏覽器 打印機
.visible-print 不可見的 可見的
.hidden-print 可見的 不可見的
  • 手機 ✔ 手機中可見
  • 平板 ✔ 平板中可見
  • PC ✔ 桌面中可見
  • 手機 ✔ 手機中不可見
  • 平板 ✔ 平板中不可見
  • PC ✔ 桌面中不可見
