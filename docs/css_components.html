AlertTip一個彈出式提醒樣式

AlertTip 有兩種呈現方式，一個是「行內」，一個是「塊級」。

  • 封裝容器只使用 .alertTip 類即顯示行內效果
  • 封裝容器在 .alertTip 類基礎上加入 .alertTip-block 類即為塊級呈現效果。
注意! 看這裡！對！看過來！好了，沒事你可以走了。 
// 行內效果

<div class="alertTip">
  <button type="button" class="close-btn" data-dismiss="alert">×</button>
  <strong>注意!</strong> 看這裡！對！看過來！好了，沒事你可以走了。
</div>

注意!

看這裡！對！看過來！好了，沒事你可以走了。

// 塊級效果

<div class="alertTip alertTip-block">
  <button type="button" class="close-btn" data-dismiss="alert">×</button>
  <h4>注意!</h4>
  <p>看這裡！對！看過來！好了，沒事你可以走了。</p>
</div>

AlertTip 額外附帶了3種主題，可用於各種提醒類型。將下面的類名加入到 alertTip 類所在標籤即可。

  • .alertTip-success 即用於「成功」內容的樣式
  • .alertTip-danger 即用于「警告」內容的樣式
  • .alertTip-info 即用於「描述」內容的樣式

Button 各種按鈕樣式

Button 模塊附帶了數種按鈕主題，可選擇配合情況使用

  • .btn 默認按鈕樣式
  • .btn .btn-success 「成功」按鈕樣式
  • .btn .btn-info 「描述」按鈕樣式
  • .btn .btn-warning 「注意」按鈕樣式
  • .btn .btn-danger 「警告」按鈕樣式
  • .btn .btn-link 「鏈接」按鈕樣式
// 默認按鈕
<button type="button" class="btn">默認</button>
// 成功按鈕
<button type="button" class="btn btn-success">成功</button>
// 描述按鈕
<button type="button" class="btn btn-info">描述</button>
// 注意按鈕
<button type="button" class="btn btn-warning">注意</button>
// 警告按鈕
<button type="button" class="btn btn-danger">警告</button>
// 鏈接按鈕
<button type="button" class="btn btn-link">鏈接</button>

另外還有兩種默認的尺寸可供直接指定

  • 默認即為標準大小
  • 如果加入 .btn-large 類則為大尺寸按鈕
// 默認尺寸
<button type="button" class="btn">默認</button>
// 大號按鈕
<button type="button" class="btn btn-large">默認</button>

不可用按鈕樣式 可以通過指定 disabled 屬性或加入 .disabled 類來實現。 

// 被禁用的按鈕
<button type="button" class="btn btn-success" disabled="disabled">成功</button>
<button type="button" class="btn btn-info disabled">描述</button>

塊級按鈕樣式 可以通過加入 .btn-block 類來實現。 

// 塊級按鈕
<button type="button" class="btn btn-block">描述</button>

將多個按鈕以一組的效果呈現，我們叫他「按鈕組」，實現方式是用一個 .btn-group 容器將多個按鈕元素包裹起來即可。

另外，如果使用 .btn-group-vertical 來包裹多個按鈕，那這個按鈕組就會以豎排的形式呈現。 

// 按鈕組
<div class="btn-group">
  <button type="button" class="btn">左</button>
  <button type="button" class="btn">中</button>
  <button type="button" class="btn">右</button>
</div>
								
// 豎排按鈕組
<div class="btn-group-vertical">
  <button type="button" class="btn">一</button>
  <button type="button" class="btn">二</button>
  <button type="button" class="btn">三</button>
</div>

將多個按鈕組組織到一起的樣式，我們稱之為「按鈕欄」，實現方式是用一個 .btn-bar 容器將多個按鈕組包裹起來。 

// 按鈕欄
<div class="btn-bar">
  <div class="btn-group">
	<button type="button" class="btn">1</button>
	...
  </div>
  <div class="btn-group">
	<button type="button" class="btn">5</button>
	...
  </div>
  <div class="btn-group">
	<button type="button" class="btn">7</button>
	...
  </div>
</div>

Forms 一套表單樣式

Input 控件

BubalusFE 為各種類型 Input 控件設計了統一的樣式，要應用 Input 控件樣式需要為其添加 .form-control 類。 

<input type="text" class="form-control" placeholder="這是一個 text 類型的 input 元素">
								
<input type="text" class="form-control" placeholder="這是一個被禁用的 text 類型的 input 元素" disabled>

Textarea 控件

BubalusFE 沒有為 Textarea 控件重置高度，而且作者建議應該通過設置換行屬性 rows 來改變 Textarea 控件的高度。
要應用 Textarea 控件樣式需要為其添加 .form-control 類。很眼熟？那是因為他和 Input 控件應用了用一個樣式類！ 

<textarea class="form-control" rows="3" placeholder="這是一個 Textarea 控件"></textarea>
								
<textarea class="form-control" rows="3" placeholder="這是一個被禁用的 Textarea 控件" disabled></textarea>

Checkbox & Radio 控件

  • 通常情況下，Checkbox 控件和 Radio 控件是以塊級樣式來佈局的，即每組控件獨佔一行。
