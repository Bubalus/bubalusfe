排版 一些内置和附加的排版類

BubalusFE 設置了全局默認文字大小爲14px，行高約為20px，並為大部分標簽重設了排版樣式以使其能在多種瀏覽器中以統一的方式呈現。

小號文本 <small> 標簽，提供了比標準字體大小還要稍小一些的字號，約為12px。

This line of text is meant to be treated as fine print.

<small>This line of text is meant to be treated as fine print.</small>

粗體文本 <strong> 標簽，爲粗體字，相當於font-weight:700。

The following snippet of text is rendered as bold text.

<strong>rendered as bold text</strong>

文本對齊 使用 .text-left .text-center .text-right 來實現文字的左、中、右對齊。

Left aligned text.

Center aligned text.

Right aligned text.

<p class="text-left">Left aligned text.</p>
<p class="text-center">Center aligned text.</p>
<p class="text-right">Right aligned text.</p>

引用 使用 <blockquote> 標簽在文檔中引用別處的資源

Lorem ipsum dolor sit amet, consectetur adipiscing elit. Integer posuere erat a ante.

<blockquote>
<p>Lorem ipsum dolor sit amet, consectetur adipiscing elit. Integer posuere erat a ante.</p>
</blockquote>

如果要在引用内容中加入引用來源，可以在引用内容的末尾插入<small>。

Lorem ipsum dolor sit amet, consectetur adipiscing elit. Integer posuere erat a ante.

Someone famous in Source Title 
<blockquote>
<p>Lorem ipsum dolor sit amet, consectetur adipiscing elit. Integer posuere erat a ante.</p>
<small>Someone famous in <cite title="Source Title">Source Title</cite></small>
</blockquote>

代碼 使用 <code> 或 <pre> 在文檔中插入代碼

行内的代碼内容可以使用 <code> 標簽

For example, <section> should be wrapped as inline. 
For example, <code><section></code> should be wrapped as inline.

多行的代碼段則應該使用 <pre> 標簽

<p>Sample text here...</p>
<pre><p>Sample text here...</p></pre>
