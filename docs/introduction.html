概述

BubalusFE 是一套 web前端開發工具集。它提供了一套CSS框架和一組基於JQuery的常用javascript組件集，以及一套實踐范例。

文件結構

所有的CSS、javascript組件和媒體内容都被存放在 assets 文件夾，這是一個比較常見的文件結構。

  • CSS/ - 用於存放項目中所有CSS文件
  • CSS/bubaluslayout - BubalusFE 的 CSS 組件
  • CSS/bubaluslayout/core - BubalusFE 核心樣式表
  • CSS/bubaluslayout/module - BubalusFE CSS模塊
  • CSS/config - 用於存放 CSS 組件的配置文件
  • CSS/project - 用於存放當前項目中用到的 CSS 文件
  • CSS/quoted - 用于存放來自於第三方的 CSS 文件
  • images/ - 用於存放項目中所有媒體文件，如圖片等
  • images/bubaluslayout - CSS 組件中用到的媒體文件
  • images/project - 用於存放當前項目中用到的媒體文件
  • js/ - 用于存放項目中所有 javascript 文件
  • js/ui - 用于存放 UI 類javascript文件
  • js/utility - 用于存放 功能 類javascript文件

CSS 組件 assets/css

BubalusFE 的CSS組件沿用 BubalusLayout CSS框架（v5.x），使用SASS進行編輯維護，提供開關式的功能設置，方便定製化編譯。
BubalusLayout 的本次升級經過徹底的重新設計，將帶來更好的性能和體驗，同時仍將爲早期的IE瀏覽器提供基礎的支援。

核心樣式表 assets/css/bubaluslayout/core

這些文件是 BubalusLayout 的核心樣式表文件，提供了樣式重置、柵格佈局、IE早期版本的支持等最重要的功能。
注意請不要修改這些文件，方便升級的同時還可防止出現一些奇怪的問題。

樣式模塊 assets/css/bubaluslayout/module

這些文件是 BubalusLayout 的樣式模塊文件，他們提供了大量常用的預置樣式。
如果你想要定製修改這些模塊，請注意，爲了方便日後升級維護，我們不建議你直接在這些文件上進行修改。
您可以在該模塊的文件夾中新建一個補丁樣式，并將其插入到 assets/css/bubaluslayout.scss 中該模塊位置的後面，并重新編譯 bubaluslayout.scss。

javascript組件 assets/js

BubalusFE 的 javascript組件，包含瀏覽器檢測組件（Modernizr）、資源異步加載組件與各種常用的UI組件和功能組件。

javascript UI 模块 assets/js/ui

各种 UI 类 javascript 模块，包含如 tabs、tooltip、dropdown 等等。

javascript 功能組件 assets/js/utility

各种功能类 javascript 模块，包含如 表单验证、动画、AJAX 等。

javascript 核心组件 assets/js/bubalusbase.js

bubalusbase.js 包含了浏览器检测、模块加载、javascript模板等最核心的功能。
