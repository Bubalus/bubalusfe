概述

BubalusFE 是一套 web前端開發工具集。它提供了一套CSS框架和一組基於JQuery的常用javascript組件集，以及一套實踐范例。

文件結構

BubalusFE 採用了一種模塊式的文件結構。

  • docs 文件夾用於存放測試案例及文檔
  • modules 文件夾用於存放 BubalusFE 的各個模塊組件
  • project 文件夾則用於存放項目中需要用到的 CSS、javascritp、圖片等文件

根目錄中的 bubalusbase.js 和 bubaluslayout.scss 兩個文件是頁面需要引用的文件，他們的使用方法將在後面專門介紹。

BubalusFE 預置了一些經常會用到的模塊，就像上面說的，他們都被存放在 modules 文件夾中。

在這些模塊裏有兩個模塊比較特殊，一個是bubalusglobal模塊，看名字就知道他的裏面裝的全是和「全局」相關的一些東西，包括一些全局的變量和方法。
另一個是 bubaluslayout 模塊。他是 BubalusFE 的 CSS 核心模塊，提供重置和佈局等核心功能。

這兩個模塊都是 BuablusFE 不可缺少的核心模塊，其他模塊大多對他倆有依賴

模塊 modules/

BubalusFE 內置了一些常用的模塊，他們提供了常用的 CSS組件 和 Javascript組件

modules 裏的這些文件夾是 BubalusFE 預置的一些模塊，一個文件夾代表一組模塊，每組模塊中包含其對應的 CSS組件 和 Javascript組件。

文檔中我們以 {moduleName} 來代表一個模塊的模塊名

  • images 文件夾用於盛放模塊用到的媒體文件
  • _mixins.scss 為 CSS組件 的方法庫
  • _variables.scss 為 CSS組件 的變量庫
  • theme.scss 為 CSS組件 的樣式主題
  • {moduleName}.scss 為 CSS組件 的主文件
  • {moduleName}.js 為 Javascript組件 的主文件
// 樣式模塊目錄結構

    {moduleName}
	│
	├─ images
	│
	├─ _mixins.scss
	│
	├─ _variables.scss
	│
	├─ theme.scss
	│
	├─ {moduleName}.scss
	│
	└─ {moduleName}.js

核心樣式表 modules\bubaluslayout

BubalusFE 內置了一套 CSS框架，使用SASS進行編輯維護，提供開關式的功能設置，方便定製化編譯。

BubalusFE 的CSS組件采用 BubalusLayout CSS框架（v5.x），經過徹底的重新設計後可以提供更好的性能和體驗，同時仍將爲早期的IE瀏覽器提供基礎的支援。
這些文件是提供了樣式重置、柵格佈局、IE早期版本的支持等最重要的功能。並且其他 樣式模塊 都或多或少的依賴於這些文件所提供的參數和方法，所以請不要修改這些文件，方便升級的同時還可防止出現一些奇怪的問題。

  • images 目錄用於存放核心樣式表用到的媒體資源
  • common.scss 一些公共樣式
  • iehacks.scss 應該不用解釋？
  • layout.scss 一套響應式柵格及方法
  • reset.scss 一套頗強力的 CSS 重置工具
// 核心樣式表目錄結構

    bubaluslayout
	│
	├─ images
	│
	├─ common.scss
	│
	├─ iehacks.scss
	│
	├─ layout.scss
	│
	└─ reset.scss

要被頁面引用的兩個文件 bubalusbase.js 和 bubaluslayout.css

其實如果算上對 早期IE版本 提供支援的 modules/bubaluslayout/iehacks.css 的話，三個文件才對。

bubaluslayout.css

bubaluslayout.css 是由 bubaluslayout.scss 生成的文件，其中包含了 核心CSS 模塊和其他模塊的 CSS組件，以及項目所需的CSS樣式。

bubalusbase.js

bubalusbase.js 包含了浏览器检测、模块加载、javascript模板等最核心的功能。
